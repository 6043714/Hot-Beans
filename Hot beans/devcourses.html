Here Are A List Of Available Spaces At Hot Beans:

-Junior Front-End Developer:
Responsible for implementing user interfaces and translating design wireframes into functional web pages using HTML, CSS, and JavaScript. Collaborates with the design team to ensure seamless integration of visuals and user interactions.

-Junior Back-End Developer:
Involved in developing server-side logic and database integration for web applications. Works with frameworks like Node.js and PHP, along with databases such as MySQL or MongoDB. Assists in building robust and scalable back-end solutions.

-Junior Full-Stack Developer:
Participates in the development of both front-end and back-end components of web applications. Collaborates with the team to ensure efficient communication between client-side and server-side code. Proficiency in languages like JavaScript, HTML, CSS, and knowledge of frameworks (e.g., React, Express) are essential.

-Junior QA Tester:
Responsible for testing and debugging web applications to ensure functionality and user experience quality. Executes test plans, reports bugs, and collaborates with the development team to address issues. Familiarity with testing frameworks and tools, such as Selenium or Cypress, is advantageous.
